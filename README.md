# dotfiles
## Usage
### Clone Project and Create Symbolic link
```
cd $HOME
git clone https://github.com/Y-Fujikawa/dotfiles
cd $HOME/dotfiles
chmod +x dotfilesLink.sh
./dotfilesLink.sh
```

### NeoBundle Install
```
mkdir -p ~.vim/bundle
git clone https://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
```
